About me

Welcome, I'm Rony Hanna, I like creating awesome software. I have significant experience in game development with particular interest in gameplay and engine development. I'm currently a Unity Developer at Kai's Clan in Auckland working on an educational AR game. I earned a bachelor's degree in Software Engineering from Media Design School. You can view a few of my personal and group projects here. Feel free to get in touch, I'd be happy to chat about job opportunities and projects I've worked on.

Projects

Titan Voyager is a first-person shooter game written in C++ and modern OpenGL (watch gameplay footage here). The game takes place on a mysterious Saturnian moon where the player has to retrieve some important data while also fighting enemies. This project captured a lot of interest on reddit and was discussed here.

Slimey Jump is a 2D platformer similar to Doodle Jump written in C++ and modern OpenGL. The player has to guide a slime up a series of platforms without falling or colliding with an enemy. The enemies are accompanied with a growling sound and can be killed by the player.

District Underground is a Third-person action game made in Unity. This project was a group project consisting of 3 artists and 3 programmers. I worked as a gameplay programmer and created a number of gameplay mechanics for the game.

An Angry Birds clone written in C++ and powered by the Box2D physics engine. This project was developed for a 2nd-year university physics paper. The demo features two simple levels, three different Box2D joints, and three different birds with different mechanics.

A small pathfinding demo I made in my free time in Unity which features different types of search algorithms. The demo features a Breadth-first traversal, Depth-first traversal, Dijkstra's algorithm, and A*.

Lava Plaza is a simple game made in Unity where the player has to dodge boulders for as long as possible and avoid falling into the lava. This was a group project consisting of 3 programmers. I worked as a gameplay programmer and implemented several gameplay mechanics for the game.
